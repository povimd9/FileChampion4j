\doxysubsection{Detailed Description}
This class is used to change the owner and permissions of a file. Set Owner uses java.\+nio.\+file.\+Files.\+set\+Owner for cross-\/platform support. Set Permissions attempts to identify the Operating System, using java.\+nio.\+file.\+attribute.\+Acl\+File\+Attribute\+View for Windows, and java.\+nio.\+file.\+attribute.\+Posix\+File\+Attribute\+View for any other.
